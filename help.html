🔑 Google Gemini APIキーについて

help なぜAPIキーが必要なのですか？

これらのアプリには、AI（人工知能）の頭脳そのものは入っていません。 あなたの代わりに問題を考えたり、プランを作成したりするために、インターネットを通じてGoogleのAIサーバー（Gemini）に接続する必要があります。

APIキーは、GoogleのAIサーバーを利用するための「通行手形」のようなものです。

APIキーを入力することで、このアプリからGoogleのAIに「問題を考えて！」「採点して！」とお願いできるようになります。
※ 入力されたキーはあなたのブラウザ内でのみ使用され、アプリの開発者（私）に送信されることはありませんのでご安心ください。

vpn_key APIキーの取得方法 (無料)

以下の手順で、Googleから無料でキーを取得できます。

1. Google AI Studioにアクセス

Googleアカウント（Gmailなど）でログインしてください。

Google AI Studioを開く open_in_new

2. キーを作成する

画面左上の [Create API key] ボタンをクリックします。
「Create API key in new project」を選択すると、すぐにキーが発行されます。

3. コピーして貼り付け

生成された Alza... から始まる長い文字列があなたのAPIキーです。
これをコピーして、アプリの入力欄に貼り付けてください。

⚠️ 注意事項
・APIキーは他人に教えないようにしてください。
・無料プランの範囲内であれば料金はかかりません。
・ブラウザを閉じると入力内容はリセットされます（セキュリティのため保存されません）。
